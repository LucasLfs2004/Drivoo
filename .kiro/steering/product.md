# Visão Geral do Produto

**Drivoo** é uma aplicação mobile React Native para realizar o match entre instrutores de autoescola e alunos que querem tirar a CNH no Brasil, focando inicialmente na capital São Paulo. O aplicativo fará um split de pagamentos, onde o aluno contrata a aula e o instrutor é responsável por dar a aula.

## Funcionalidades Principais
- **Sistema multi-usuário**: Suporte para clientes (alunos) e administradores
- **Gerenciamento de aulas**: Agendamento, cronogramas e informações das aulas
- **Integração de pagamentos**: Processamento de pagamentos via Stripe com split payment
- **Gerenciamento de instrutores**: Suporte para múltiplos instrutores credenciados
- **Perfis de usuário**: Gerenciamento de perfis de alunos e administradores
- **Dashboard analítico**: Gráficos visuais e relatórios para administradores
- **Sistema de chat**: Comunicação entre instrutores e alunos

## Tipos de Usuário
- **Alunos (Clientes)**: Visualizam instrutores próximos, agendam aulas, fazem pagamentos, conversam com instrutores
- **Instrutores**: Gerenciam disponibilidade, recebem pagamentos, se comunicam com alunos
- **Administradores**: Gerenciam a plataforma, visualizam analytics, supervisionam operações

## Requisitos Específicos para Clientes
- **Listagem de instrutores**: Exibição de instrutores próximos com detalhes e disponibilidade de agenda
- **Exibição de valores**: Tela clara com preços das aulas
- **Finalização de reserva**: Área de checkout com confirmação da compra
- **Split payment**: Integração com componentes do Stripe para divisão de pagamentos
- **Chat integrado**: Área de comunicação entre instrutores e alunos

## Domínio de Negócio
O app conecta pessoas que querem tirar a CNH com instrutores credenciados pelo DETRAN, fornecendo uma plataforma completa para agendamento de aulas, processamento de pagamentos e comunicação entre as partes.
